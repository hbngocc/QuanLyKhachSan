# Quản lý khách sạn 

## Giới thiệu
Ứng dụng quản lý khách sạn được viết bằng C# Windows form, sử dụng SQL Server để làm cơ sở dữ liệu 

## Tính năng chính 
- Đăng ký khách hàng
- Quản lý phòng
- Trả phòng 
- Thêm/Sửa/Xóa nhân viên

## Công nghệ sử dụng 
- C# Windows form (.NET Framework)
- SQL Server
- Visual Studio 2022
